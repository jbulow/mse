
= Text support =

* (font3d "name" ptsize 'bold 'italic ['underscore | 'underline] 'strikeout)
** Creates a font object and gets all the glyphs, ready to be used by OpenGL.

* (F "text")
** F is a font object created using font3d.
** Renders the given ascii text using the given font.
** Behaves like format
